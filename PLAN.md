# PLAN.MD — Декомпозиция проекта “Android App Store MVP”

## Описание проекта

Цель проекта — разработать MVP-приложение "App Store" с использованием современных Android-технологий:

- Jetpack Compose для UI  
- Navigation Compose  
- Room для локального хранения данных  
- MVVM архитектура  
- Kotlin 2.x  
- Единый список приложений и экран карточки приложения  
- Возможность установки и удаления приложений (через изменение состояния)

Проект включает этапы проектирования, разработки, настройки инфраструктуры и финальной подготовки к публикации.
