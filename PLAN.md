# PLAN.MD — Декомпозиция проекта "Android App Store MVP"

## Описание проекта

Цель: разработать MVP магазина приложений на Android.  
Приложение должно предоставлять:

- список приложений;
- карточку приложения;
- установку и удаление приложений.
